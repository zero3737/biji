## 功能模块说明

本项目基于 Hexo 博客框架快速搭建。

* 文章相关：创建、分类、标签、归档、搜索、近期文章模块、加密、文章置顶
* 支持 RSS 订阅
* 3D Tag Cloud
* 在线留言功能
* 评论功能
* 统计相关：网站流量分析、字数、评论数、文章阅读次数统计
* 第三方分享：微信、QQ、微博、豆瓣、邮箱
* 细节功能：图片查看、代码块复制
* 看板娘
